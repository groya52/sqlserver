DAX - это функциональный язык, специально разработанный для расчета бизнес-формул по модели данных.

Можно написать выражение DAX для определения следующих объектов:
    - Измерение (Measure). 
        Выражение DAX оценивается в контексте фильтра, где строки одной или нескольких таблиц в модели данных агрегируются и фильтруются в соответствии с неявными фильтрами, определенными в контексте фильтра.
    - Расчетный стобец (Calculated column).
        Выражение DAX оценивается для каждой строки таблицы, а синтаксис DAX может неявно ссылаться на значение каждого столбца таблицы. 
        Результат вычисляемого столбца сохраняется в модели данных и автоматически обновляется каждый раз, когда в любой таблице модели данных выполняется операция обновления.
    - Расчетная таблица (Calculated table).
        Выражение DAX возвращает таблицу, которая сохраняется в модели данных, и она автоматически обновляется каждый раз, когда в модели данных выполняется операция обновления в любой таблице.
    - Запрос (Query).
        DAX-выражение возвращает таблицу, которая материализуется в результате самого запроса.
        
Синтаксис DAX.
    Формат ссылки на столбец.
        
        'TableName'[ColumnName]
        
        Можно не указывать имя таблицы, если ссылаеться на столбец или меру в той же таблице, где определяеся формула.
        
        Можно опустить одинарные кавычки, если имя таблицы не начинается с числа, не содержит пробелов и не является зарезервированным словом.

    Рекомендации.
        Всегда указывайте имя таблицы для ссылки на столбец. 

            'TableName'[ColumnName]

        Всегда опускайте имя таблицы для ссылки на меру.

            [MeasureName]
            
Типы данных DAX.
    int64
        Целое число. 
        Типы данных Т-SQL bigint, int, smallint, и tinyint.
    double
        Десятичное число.
        Типы данных Т-SQL float и real.
    decimal 
        Валюта. Фиксированное десятичное число, с четырьмя десятичными цифрами фиксированной точности.
        Типы данных Т-SQL smallmoney.
    dateTime
        Дата и время.
        Типы данных Т-SQL money и date, datetime, datetime2, smalldatetime, и time.
    boolean
        Логическое значение.
        Типы данных Т-SQL bit.
    string
        Строка (Unicode).
        Типы данных Т-SQL char, varchar, text, nchar, nvarchar, и ntext.
    binary
        Большой двоичный объект, также известный как BLOB.
        Типы данных Т-SQL binary, varbinary и image.
        
    DAX имеет мощную систему обработки типов, когда вы пишете выражение DAX, результирующий тип основан на типе терминов, используемых в выражении.

COUNT 
    Работает только с числовыми столбцами.
COUNTA
    Работает с любыми типами столбцов.
COUNTBLANK
    Возвращает количество пустых ячеек в столбце.
COUNTROWS
    Возвращает количество строк в таблице.
DISTINCTCOUNT
    Возвращает количество различных значений столбца.

SQLServer:Databases 
    https://technet.microsoft.com/en-us/library/ms189883(v=sql.110).aspx
    
    Active Transactions
        Количество активных транзакций в базе данных.
    Backup/Restore Throughput/sec
        Пропускная способность операций чтения и записи при резервном копировании или восстановлении базы данных.
    Bulk Copy Rows/sec
        Количество строк, скопированных за секунду при выполнении операции массового копирования.
    Bulk Copy Throughput/sec
        Объем данных, скопированных при операции массового копирования в секунду (в килобайтах).
    Commit table entries
        Размер находящейся в памяти части таблицы фиксации для базы данных.
    Data File(s) Size (KB)
        Совокупный размер всех файлов данных базы данных.
    DBCC Logical Scan Bytes/sec
        Количество байтов, считанных за секунду операциями логического просмотра при выполнении команд консоли базы данных (DBCC).
    Log Cache Hit Ratio
        Процент операций чтения журнала, получивших данных из кэша журнала.
    Log Cache Reads/sec
        Количество операций чтения, выполненных за секунду из кэша менеджера журнала.
    Log File(s) Size (KB)
        Совокупный размер (в килобайтах) всех файлов журнала транзакций в базе данных.
    Log File(s) Used Size (KB)
        Совокупный размер используемого пространства во всех файлах журнала базы данных.
    Log Flush Wait Time
        Совокупное время ожидания записи журнала на диск (в миллисекундах).
    Log Flush Waits/sec
        Число операций фиксации в секунду, ожидающих записи журнала на диск.
    Log Flush Write Time (ms)
        Время в миллисекундах для выполнения операций записи при сбросе журнала на диск, которые были выполнены в последнюю секунду.
    Log Flushes/sec
        Число записей журнала на диск в секунду.
    Log Growths
        Общее число выполнений операции расширения журнала транзакций базы данных.
    Log Shrinks
        Общее число выполнений операции сжатия журнала транзакций базы данных.
    Log Pool Cache Misses/sec
        Количество запросов, при выполнении которых блок журнала отсутствовал в пуле журналов.
    Log Pool Disk Reads/sec
        Число операций чтения диска, которые выдал пул журнала для получения блоков журнала.
    Log Pool Requests/sec
        Количество запросов блока журнала, обрабатываемых пулом журнала.
    Log Truncations
        Количество сжатий журнала транзакций.
    Percent Log Used
        Процент используемого пространства журнала.
    Repl. Pending Xacts
        Число транзакций в журнале транзакций базы данных публикации, которые отмечены для репликации, но еще не доставлены в базу данных распространителя.
    Repl. Trans. Rate
        Число транзакций, читаемое за секунду из журнала транзакций базы данных публикации и помещаемое в базу данных распространителя.
    Shrink Data Movement Bytes/sec
        Объем данных, перемещаемых за секунду операциями автосжатия или инструкциями DBCC SHRINKDATABASE или DBCC SHRINKFILE.
    Tracked transactions/sec
        Количество зафиксированных транзакций, записанных в таблицу фиксации для базы данных.
    Transactions/sec
        Количество начатых в базе данных за секунду транзакций.
    Write Transactions/sec
        Число транзакций, которые производили запись в базу данных и были зафиксированы, за последнюю секунду.

    Auto-Param Attempts/sec
        Число попыток автоматической параметризации, выполненных за секунду.
    Batch Requests/sec
        Число пакетов команд Transact-SQL, полученных за секунду. 
    Failed Auto-Params/sec
        Число ошибочных попыток автоматической параметризации, выполненных за секунду.
    Forced Parameterizations/sec
        Число успешных принудительных параметризаций в секунду.
    Guided Plan Executions/sec
        Число выполнений плана в секунду, когда план запроса создается с помощью руководства плана.
    Misguided Plan Executions/sec
        Количество планов в секунду, в которых руководство плана не может быть выполнено во время формирования плана.
    Safe Auto-Params/sec
        Число попыток надежных автоматических параметризаций в секунду.
    SQL Attention rate
        Количество запросов клиента на завершение текущего запроса.
    SQL Compilations/sec
        Число компиляций SQL, выполненных за секунду.
    SQL Re-Compilations/sec
        Число повторных компиляций инструкций в секунду.
    Unsafe Auto-Params/sec
        Число ненадежных попыток автоматической параметризации, выполненных за секунду.
    
SQLServer:Transactions
    https://technet.microsoft.com/en-us/library/ms189038(v=sql.105).aspx

    Free Space in tempdb (KB)
        Объем свободного пространства в базе данных tempdb.        
    Longest Transaction Running Time
        Время (в секундах) с момента запуска транзакции, которая оставалась активной дольше любой другой текущей транзакции.
    NonSnapshot Version Transactions
        Число активных в настоящий момент транзакций, не использующих уровень изоляции моментальных снимков, но производящих изменения данных, которые привели к формированию версий строк в хранилище версий tempdb.
    Snapshot Transactions
        Число активных в настоящий момент транзакций, которые используют уровень изоляции моментальных снимков.
        
        Счетчик объекта Snapshot Transactions возвращает данные о том, когда произошел первый доступ к данным, а не когда была выполнена инструкция BEGIN TRANSACTION.
    Transactions
        Число активных в настоящее время транзакций любого типа.
    Update conflict ratio
        Процент транзакций, использующих уровень изоляции моментальных снимков, которые столкнулись с конфликтами обновления за последнюю секунду.
        
        Конфликт обновления возникает, когда транзакция уровня изоляции моментальных снимков пытается изменить строку, 
        которая в последний раз была изменена другой транзакцией, незафиксированной на момент запуска транзакции уровня изоляции снимков.
    Update Snapshot Transactions
        Число активных в настоящий момент транзакций, которые используют уровень изоляции моментальных снимков и содержат измененные данные.
    Version Cleanup rate (KB/s)
        Скорость удаления версий строк из хранилища версий изоляции моментальных снимков в базе данных tempdb.
    Version Generation rate (KB/s)
        Скорость добавления новых версий строк в хранилище версий изоляции моментальных снимков в базе данных tempdb.   
    Version Store Size (KB)
        Объем пространства в базе данных tempdb, используемого для хранения версий строк уровня изоляции моментальных снимков.
    Version Store unit count
        Число активных единиц распределения в хранилище версий изоляции моментальных снимков в базе данных tempdb.
    Version Store unit creation
        Число единиц распределения, созданных в хранилище изоляции моментальных снимков с момента запуска экземпляра компонента Database Engine.
    Version Store unit truncation
        Число единиц распределения, удаленных из хранилища изоляции моментальных снимков с момента запуска экземпляра Database Engine.

SQLServer:Memory Manager
    https://technet.microsoft.com/en-us/library/ms190924(v=sql.110).aspx
    
    Free Memory (KB)
        Количество памяти, которую сервер не использует в настоящий момент.
    Granted Workspace Memory (KB)
        Определяет общий объем памяти, предоставляемой в настоящее время для выполнения процессов, таких как операции хэша, сортировки, массовой копии и создания индекса.
    Target Server Memory (KB)
        Идеальный объем памяти, который может потреблять сервер.
    Total Server Memory (KB)
        Объем памяти, потребляемой сервером.
    Database Cache Memory (KB)
        Определяет объем памяти, который сервер использует в данный момент для кеша страниц базы данных.
    Stolen Server Memory (KB)
        Указывает объем памяти, который сервер использует для целей, отличных от страниц базы данных.
    Reserved Server Memory (KB)
        Указывает объем памяти, который сервер зарезервировал для будущего использования.
    Memory Grants Outstanding
        Указывает общее количество процессов, успешно получивших грант памяти рабочей области.
    Memory Grants Pending
        Указывает общее количество процессов, ожидающих выделения памяти рабочего пространства.
        
        Другой способ получить это значение это использовать DMV sys.dm_exec_query_memory_grants. Значение null в столбце grant_time указывает, что процесс ожидает выделения памяти.
    Lock Memory (KB)
        Указывает общий объем памяти, которую сервер использует для блокировок.
    Optimizer Memory (KB)
        Определяет общий объем памяти, используемый сервером для оптимизации запросов.
           
SQLServer:Buffer Manager
    https://docs.microsoft.com/en-us/sql/relational-databases/performance-monitor/sql-server-buffer-manager-object
    
    Page reads/sec
        Показывает количество физических страниц базы данных, считанных SQL Server.
        
        Устранение неполадок с чрезмерными чтениями страниц потребует дополнительной информации, например, выяснение, 
        какие запросы выполняют большое количество чтений или анализ вывода некоторых DMV, 
        таких как sys.db _io_virtual_file_stats или sys.dm_db_index_usage_stats.
    Page writes/sec
        Показывает количество физических записей страниц базы данных, выданных в секунду.
    Page Life Expectancy
        Показывает среднее количество секунд, в течение которых страница данных будет оставаться в пуле буферов.
        Показывает общее значение для всех узлов NUMA.
        
        Если используется архитектура NUMA, вы должны использовать PLE из объекта SQLServer:Buffer Node.
    Buffer cache hit ratio
        Показывает процент страниц, которые были возвращены из пула буферов без необходимости обращения к диску.
    Checkpoint pages/sec
        Количество страниц, сброшенных на диск в секунду, либо с помощью контрольной точки, либо любой другой операции, требующей, чтобы все грязные страницы были сброшены на диск. 
    Lazy writes/sec
        Число буферов, записываемых за одну секунду модулем отложенной записи.
        
SQLServer:Latches
    https://technet.microsoft.com/en-us/library/ms177421(v=sql.110).aspx
     
    Average Latch Wait Time (ms)
        Средняя длительность ожидания кратковременной блокировки запроса (в миллисекундах).
    Latch Waits/sec
        Количество запросов на кратковременную блокировку, при которых блокировка не была предоставлена немедленно.
    Number of SuperLatches
        Текущее количество кратковременных блокировок по схеме SuperLatch.
    SuperLatch Demotions/sec
        Количество блокировок по схеме SuperLatch, пониженных до обычных кратковременных блокировок в течение последней секунды.
    SuperLatch Promotions/sec
        Количество кратковременных блокировок по схеме SuperLatch в течение последней секунды.
    Total Latch Wait Time (ms)
        Общая длительность ожидания (в миллисекундах) после запросов кратковременной блокировки в течение последней секунды.
       
SQLServer:Locks
    https://technet.microsoft.com/en-us/library/ms190216(v=sql.110).aspx
    
    Average Wait Time (ms)
        Средняя длительность ожидания (в миллисекундах) для всех запросов блокировки, при которых потребовалось ожидание.
    Lock Requests/sec
        Количество новых блокировок и преобразований блокировок за секунду, запрошенное у диспетчера блокировок.
    Lock Timeouts (timeout > 0)/sec
        Количество запросов блокировок в секунду, время ожидания которых истекло, кроме запросов блокировок NOWAIT.
    Lock Timeouts/sec
        Количество запросов блокировок в секунду, время ожидания которых истекло, включая запросы блокировок NOWAIT.
    Lock Wait Time (ms)
        Суммарное время ожидания (в миллисекундах) блокировок за последнюю секунду.
    Lock Waits/sec
        Количество запросов блокировок в секунду, которые потребовали ожидания участника.
    Number of Deadlocks/sec
        Количество запросов блокировок за секунду, которые завершились взаимоблокировками.
 
SQLServer:Access Methods
    https://technet.microsoft.com/ru-ru/library/ms177426(v=sql.105).aspx

    FreeSpace Scans/sec
        Количество операций сканирования в секунду, выполненных для поиска свободного пространства на страницах, 
        выделенных единице распределения, для вставки или изменения фрагмента записи.
    Full Scans/sec
        Количество неограниченных операций полного сканирования в секунду.
        К таким операциям относятся сканирование основной таблицы и полное сканирование индекса.
    Table Lock Escalations/sec
        Показывает, сколько раз блокировки таблиц были укрупнены до гранулярности TABLE или HoBT.
    Worktables Created/sec
        Количество созданных рабочих таблиц в секунду. 
        Например рабочие таблицы можно использовать для хранения временных результатов для очереди запросов, переменных LOB и XML и курсоров.

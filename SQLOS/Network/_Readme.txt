Инструменты мониторинга.
    - Performance Monitor.
        Network Interface (Network card)
            Bytes Total/Sec
                Вы можете использовать счетчик Bytes Total/Sec, чтобы определить, как работает сетевая плата (NIC) или сетевой адаптер.
                Bytes Total/Sec должен сообщать о высоких значениях, чтобы указать большое количество успешных передач. 
                
                Сравните это значение с показателем счетчика Network Interface\Current Bandwidth, который отражает пропускную способность каждого адаптера.
        Network Segment
            % Net Utilization
                Счетчик% Net Utilization представляет процент используемой полосы пропускания сети в сетевом сегменте.

Решения для устранения узких мест в сети.
    - Оптимизация нагрузки приложения.
        Чтобы оптимизировать сетевой трафик между приложением базы данных и сервером базы данных, внесите следующие изменения в приложение:
            - Группируйте запросы нескольких баз данных в одну хранимую процедуру. 
              Затем требуется только один запрос базы данных по всей сети для набора SQL-запросов, реализованных в хранимой процедуре.
            - Не запрашивайте столбцы таблицы, которые не используются в логике приложения.
            - Перемещение бизнес-логики с интенсивным использованием данных в базу данных в виде хранимых процедур или триггеров баз данных для сокращения передачи данных по сети.
            - Если данные не часто меняются, попробуйте кэшировать информацию в приложении.
            - Минимизируйте сетевые вызовы, например, верните несколько наборов результатов.
            
              Общей проблемой является результат, возвращаемый SQL Server, который включает в себя количество строк каждого оператора. 
              Вы можете отключить это, используя SET NOCOUNT ON в верхней части вашего запроса.

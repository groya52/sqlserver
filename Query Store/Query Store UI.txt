Regressed Queries
Отображает запросы, для которых метрики выполнения ухудшились в ближайшее время.
Можно использовать это представление чтобы связать проблемы возникшие в вашем приложении с запросами, которые требуется ускорить.

Top Resource Consuming Queries
Можно выбрать необходимую метрику выполнения и определить запросы, которые имеют наибольшие значения этой метрики за выбранный интервал.
Используйте это представления, чтобы сфокусировать ваше внимание на запросах, которые потребляют наибольшее количество ресурсов.

Tracked Queries
Отслеживание интересующих запросов в реальном времени.
Если план был форсирован, то это представление можно использовать для контроля стабильности производительности выполнения запроса.

Overall Resource Consumption
Анализ общего потребления ресурсов базой данных для любой метрики выполнения.
Используйте это представление, чтобы определить шаблоны потребления ресурсов (например, дневная / ночная нагрузка) и оптимизировать потребление ресурсов в целом.

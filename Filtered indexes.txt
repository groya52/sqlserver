Отфильтрованные индексы
    Поддерживаются только простые фильтры (оператор OR не поддерживается, и вы не можете ссылаться на функции и вычисленные столбцы).
    SQL Server не может использовать отфильтрованный индекс, когда план выполнения должен быть кэширован.
    SQL Server обновляет статистику по ним, но не учитывает изменения столбцов от фильтра до порога обновления статистики. 
        Вы должны регулярно обновлять статистику по отфильтрованным индексам, когда столбцы фильтра являются изменчивыми и не включаются в индексный ключ.
        
        
Оценочная оценка мощности (70), независимость предикатов запроса друг от друга
    (Selectivity of first predicate * Selectivity of second predicate) * (Total number of rows in table)
